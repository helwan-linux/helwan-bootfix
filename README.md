## ✨ Key Features

### 🚀 1. Boot Fix (The Lifesaver)
* **What it does:** Fixes the "Missing Operating System" or "Grub Error" screens.
* **When to use it:** If your computer won't reach the login screen or if Helwan Linux disappeared from the boot menu.
* **Result:** Restores your bootloader and ensures your hardware talks to your software perfectly.

### 🧹 2. Recovery Mode (The System Tuner)
* **What it does:** Cleans and repairs the internal "plumbing" of your system.
* **When to use it:** * If you get errors like "Database is locked" during updates.
    * If apps refuse to open or crash unexpectedly.
    * If your system feels sluggish or your disk is mysteriously full.
* **Result:** Clears temporary junk, unlocks the package manager, and repairs broken system links.

### 🔑 3. Chroot Helper (The Backdoor Access)
* **What it does:** Allows you to "enter" your system even if it’s completely broken.
* **When to use it:** For advanced rescue missions—like resetting a forgotten password or fixing critical system files from a Live USB.
